# A bot's life

Another life simulation game

## Resolve dependencies

```bash
npm install
```

## Transpile typescript and watch for changes

```bash
bin/tsc -w
# or
./node_modules/typescript/bin/tsc -w
```

## Launch http server

```bash
bin/serve
# or
./node_modules/http-server/bin/http-server
```
Navigate to http://localhost:8080
